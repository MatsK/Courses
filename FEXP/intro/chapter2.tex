\section{レポートの体裁について}\label{sec:ch2}

基礎実験では、レポートの体裁を以下のように統一する。
\begin{itemize}
\item レポート用紙はA4版で横罫線の入ったものを使う。

\item ペン書き、鉛筆書き、ワープロの使用の可・不可は学科によっ
て異なるので指示に従うこと。

\item コピーの貼り付けは禁止とする。

\item 表紙とともに左上をホッチキスで留めて提出する。
\end{itemize}

\section{採点と再提出について}

採点は完成したレポートに対して行われる。未完成のまま提出して
も未提出扱いになる。また、レポートの内容によっては再提出を指
示されることがある。再提出レポートではどこが加筆訂正した部分
なのか分かりやすいようにしておくこと。

\section{レポートの構成}

実験レポートの目的は、「どんな実験をしたか」「どんな結果が得
られたか」「それについてどう考えたか」を読み手にわかりやすく
伝えることである。したがって、レポートは明解な文章で書かれ、
かつ記述の展開がスムーズに伝わるよう構成されている必要がある。
基礎実験では、一般の技術論文にならい、以下に示す5つの項目に
沿ってレポートをまとめる。
\begin{enumerate}
\item 目的
\item 理論または原理 
\item 実験の方法
\item 実験結果
\item 考察
\end{enumerate}

以下に各章の書き方を説明する。
\begin{enumerate}
\item 目的

実験の目的を簡潔にまとめる。他のすべての章にかかわる重要な部分である。

\item 理論または原理

実験の目的を達成するために用いた理論または原理をまとめる。

\item 実験の方法

具体的な実験装置や測定方法について記述する。前項までで、実験方法がすでに示されていればこの章は無くても良い。
\item  実験結果

この章では実験で得られた「事実」を以下の点に留意して書く。

  \begin{enumerate}
  \item データや解析結果はなるべく表にまとめて見やすくするよ
  う工夫する。また、どんな条件で得られた実験結果なのかを明示
  する。
  \item 目的量を求めるために用いた計算過程は、計算の流れが分
  るように示す。また、有効数字(有効桁)にも気をつけること。
  \item 物理量の名称や単位を必ず書く。単位は5.01~[V]のように
    四角カッコで囲む。単位は国際単位系(SI)を使用する。(付録
    の物理単位を参照のこと。)
\end{enumerate}

\item 考察

考察とは、行った実験について自発的に問題提起し、それに対し自
分で解答を考えるものである。何を考察するかは各自に委ねられて
いる。始めからすべて自分で考えるのは難しいので、基礎実験では
考察課題を設けているテーマが多い。これらの課題をベースにして
考察の仕方を習得する。また、感想は考察の章には書かない。も
し書きたければ別に項目を設けて書く。

\end{enumerate}

\subsection{摸式図などの描き方}

レポートを分かりやすくするためには装置の模式図や概念図、回路
図などを入れるとよい。これらの図は定規等を使ってきれいに描く。
曲線部分もテンプレートを使用するなどしてできるだけなめらかに
描く。そして、図の下部に図の通し番号と題名を書く。

\subsection{グラフを描く際の注意点}
実験結果を最も端的に表すのがグラフである。それゆえ、その意味
するところが一目で理解できるように描かなければいけない。 図1
に例として電流-電圧特性のグラフを示す。
\begin{enumerate}
\item グラフ用紙には専用の方眼紙(市販されているA4版の1mm方眼、片対数、両対数グラフ用紙)を用いる。
\item 下部に図の通し番号と題名を書く。グラフは図の一種である
  ので、グラフ1という表記はしないで、図1という書き方をする。
\item 縦・横の座標軸と目盛を書く。目盛には数値を示す。
\item 縦軸・横軸に物理量の名称と単位を座標軸のほぼ中央に明記する。
\item 測定点は、●○×等のマークでプロットする(点を打つことをプロットする、という)。
\item 測定点を曲線で結ぶ場合はなめらかに結ぶ。
\item 同じ測定を条件を変えて行う場合、一般的には図1のように
同じ図に重ね描きし、各実験条件を
図中に記入した方が分かりやすい。
\end{enumerate}


\subsection{直線化の工夫}

二つの物理量が直線関係にない時にも、縦軸と横軸の取り方を工夫
することにより、測定結果を直感的に判断しやすい直線的な関係と
して表示することができる。例を次に示す。

\paragraph{金属の比熱}
銅のような金属の低温比熱$C$は、絶対温度$T$に比例する自由電子
の寄与と、$T$の3乗に比例するイオンの熱振動の寄与の和である。
\begin{equation}
 C = \gamma T + \beta T^3
\end{equation}
この式の係数$\gamma$と$\beta$は電子やイオンの情報を含む
重要な物質定数である。いろいろな温度における比熱の測定値から
$\gamma$と$\beta$の値を決めたい。両辺を$T$で割ると
\begin{equation}
 \frac{C}{T}= \gamma + \beta T^2
\end{equation}
となり、縦軸に$C/T$をとり、横軸に$T$の2乗をとれば直線的な関
係になる。最小二乗法を用いて切片と傾きに相当する$\gamma$と
$\beta$を求めることができる。

\paragraph{超伝導体の臨界磁場}

ある種の金属はある決まった温度$T_{\rm c}$以下で電気
抵抗ゼロの超伝導状態に転移する。しかし、外部磁場を印加すると、
ある磁場$H_{\rm c}$で再び常伝導状態に転移する。$H_{\rm c}$は
温度の関数で臨界磁場といわれ、その温度依存性は
\begin{equation}
  H_{\rm c}(T)=H_{\rm c}(0)\left\{1-\left(T\over T_{\rm c}\right)^2\right\}
\end{equation}
で与えられる。温度$T$を変えながら$H_{\rm c}$を測定したデータ
から、縦軸に$H_{\rm c}$を、横軸に$T^2$をとったグラフを作れば
直線となり、これに最小二乗法を適用すれば$H_{\rm c}(0)$と
$T_{\rm c}$を求めることができる。


\subsection{対数グラフ}

縦軸だけを常用対数の目盛に取ったグラフを片対数グラフといい、
横軸・縦軸ともに常用対数の目盛を取ったグラフを両対数グラフという。

\paragraph{片対数グラフ}
縦軸の大きな区切れが等間隔になっている。それらの区切れを1区
画として常用対数を目盛ったグラフである。

図2のようにある区切れを$1=10^0$ と決めるとその一つ下は 
$0.1=10^{-1}$ となり、$1=10^0$ の一つ上は $10=10^1$ となる。
大きな区切れ1区画の間は 
$\log 2=0.301$, $\log 3=0.477$, $\log4=0.602$・・・という値を目盛る。

たとえば、$y=b{\rm e}^{ax}=b \exp(a x)$の関係があるときには
$\log y = a x \log {\rm e}+
\log b$であるので、片対数グラフを使って表示すれば直線となっ
て分かりやすくなる。つまり片対数グラフでは($x$, $y$)のプロッ
トの代りに($x$, $\log y$)をプロットしていることになる。この
ときに($x$, $\log y$)について最小二乗法を適用すれば$a$と$\log
b$を求めることができる。

\paragraph{両対数グラフ}
横軸・縦軸ともに常用対数の目盛を取ったグラフを両対数グラフ
という。

たとえば、$y=b x^a$の関係があるときには$\log y = a \log x +
\log b$であるので、両対数グラフを使って表示すれば直線となっ
て分かりやすくなる。つまり両対数グラフでは($x$, $y$)のプロッ
トの代りに($\log x$, $\log y$)をプロットしていることになる。
このときに($\log x$, $\log y$)について最小2乗法を適用すれば
$a$と$\log b$を求めることができる。

\section{文章を書くときの注意}

学生のレポートによく見られる不適切な表現をいくつか挙げておく。
レポートを書く際に気をつけること。
\begin{enumerate}
\item 「$\sim$できる\underline{と思われる}。$\sim$分かりやす
  くなる\underline{と考えられる。}」

日本語文章では若干曖昧に書く方が普通であるが、レポートでは曖
昧な表現は良くない。上記の表現はひじょうに多くレポートで見ら
れるが、できるだけ使わないようにする。具体的には「と思われる。」
「と考えられる」
という部分を消して断定した表現にして、問題が無ければ断定表現
にする。上記の場合には無くてもいいことが分かるだろう。

\item 「理論値と実験値が近い値になったので\underline{この実
験は正しい}」

実験で得られた結果は事実であり、正しい、正しく
ないと論じること自体おかしい。また理論値が信頼できるとも限
らない。この例では、実験の精度・確度を考慮して、
実験方法と結果、あるいは理論値の妥当性を論ずるべき
である。
\item 「誤差は0.01 [m]だったので\underline{充分に小さい}と思われる」

実験結果の数値だけを見て、主観的に「大きい」「小さい」と言っても無意味である。実験では「$\sim$と比較して大きい」のように客観的表現が求められる。
\item 「ソレノイドの磁界を測定すると\underline{$B=aI$}の関係から$\cdots$」

いきなり$B=aI$と書かれても、読者には$B$, $a$, $I$が何を意味するか分からない。この例のように\underline{変数や定数を定義なしに使ってはいけない}。
\item その他注意する点
\begin{enumerate}
\item 主語・述語や修飾関係に気をつけ明確な文章を書く。
\item 指示代名詞は具体的に何を指しているのかはっきり分かる場合のみ使う。
\item 必要以上に数式を書かない。実験レポートは、数学の試験ではないので式変形はどのようにしたか分かる最低限度にとどめる。
\item 実験で求める数値にはほとんど単位があるので必ず書くこと。
  単位は括弧で囲む。例えば、「電圧 15[V]」というように記述す
  る。単位およびその表記法については付録を参照のこと。
\item 第3章の「測定とデータの扱い方」でも説明するが、誤差を
  記述する場合、最確値と誤差の位を合わせる。また、誤差は、原
  則として1桁とする。

(例) 最確値5.012~[V]、誤差0.33~[V]という結果が得られたとき、電
圧$5.0\pm 0.3$~[V]のように報告する。(誤差が小数点以下第1位な
ので、最確値もそこまで書く。)
\end{enumerate}
\end{enumerate}
