\section{抵抗器}\label{sec:1}

電子回路でよく使用する部品に抵抗器がある。単位は[$\Omega$]を
用いる。

\subsection{抵抗器の材質と定格電力}                       

一般に、電子回路で使われる抵抗器は定格電力$1/8$から数ワット
程度であり抵抗部の材質により次のような物がある。
\begin{itemize} 
 \item 金属被膜：温度係数が大変小さく，精度が高く，またノイズも小さいが値段が高い。
 \item 酸化金属被膜：温度係数・精度は金属被膜より劣るが耐熱性に優れる。
 \item 炭素皮膜：低精度ですべての面で金属被膜に劣り経年劣化
   もあるが値段が安い。
\end{itemize} 

定格電力とは、例えば0.5~[W], 100~[$\Omega$]だと電力
$=I^{2}R=V^2/R$なので7~[V]の電圧を印加でき0.07~[A]の
電流を流すことができる、と言うことを意味する。但し、実際はこ
れよりも少ない電圧・電流値で使用するのが望ましい。

\section{コンデンサ}

\begin{figure}[htbp]
  \centering
  \includegraphics{condenser.eps}
  \caption{コンデンサの概観と容量の表示例}
\end{figure}
コンデンサとは2つの電極間に絶縁体(誘電体とも言う)を挿入
したもので電荷を溜めることができる。単位は[F](ファラッ
ド)を用いる。

\subsection{コンデンサの種類}

コンデンサは絶縁体の種類によって次のように分類される。

\begin{itemize}
 \item セラミック：絶縁体にセラミックを使用した物で、高周波特性が良い。無極性。
 \item フィルム：絶縁体にフィルムを使用した物で、材質は、ポ
 リエステル(マイラとも言う)、ポリプロピレン、ポリスチレン等
 がある。材質によって特性は異なるが、一般的には、絶縁抵抗値
 が高く、温度特性がよい。無極性。
\item 電解：電気分解による酸化被膜を絶縁体として使う物で、電極の金属によりアルミ電解とタンタル電解がある。大容量であるが、周波数特性、温度特性が悪く絶縁抵抗値も小さい。極性があるので接続には注意が必要である。
\item その他には、絶縁体にマイカ(雲母)、ガラス等を使用した物がある。
\end{itemize}

\subsection{表示値の読み方}

コンデンサにも静電容量等が表示されている。以下に表記例を示す
が、製品によってはこれらの情報が表示されていない物もある。図
2に外観を数例示す。221Jは$22\times10^1$~[pF] $\pm 5$\%を意味
する。左から2番目までの数字は容量の数値を示し3番目の数字が10
の乗数を表す。この表示の場合、単位の数値はp(ピコ)
$=10^{-12}$である。最後のJは誤差値表示であり、表2にまとめる。
BからNまではプラス側とマイナス側の誤差が等しく、PからZでは異
なる。大きな容量のコンデンサは記載できる面積が大きいので分か
りやすい表示をしている物が多い。電圧は耐電圧値を表す。電解コ
ンデンサは極性があるので接続の表示をしている。図のように通常
マイナス側の表示をしていて、接続線は普通マイナス側が短い。
